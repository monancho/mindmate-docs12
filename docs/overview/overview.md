# 프로젝트 개요

## 1. 서비스 한 줄 설명

MindMate는 **감정 일기 + AI 해석 + 시각화**를 통해  
사용자가 자신의 감정 패턴을 이해하고 관리할 수 있게 돕는 서비스입니다.

## 2. 해결하고자 한 문제

- 감정을 기록해도 **나중에 어떻게 해석해야 할지 모르는 문제**
- 일기 기록이 계속 쌓여도 **패턴이나 변화가 잘 보이지 않는 문제**
- 단순 할 일 관리가 아니라, **마음 상태와 리듬 중심의 자기 관리 도구가 부족한 점**

## 3. 타겟 사용자

- 감정 기복이 크다고 느끼거나, 자기 상태를 객관적으로 보고 싶은 사람
- 하루 흐름(수면, 활동, 기분)을 기록하며 **리듬을 관리**하고 싶은 사람
- “오늘 하루를 숫자/그래프/캐릭터로 정리”하는 걸 좋아하는 사용자

## 4. 주요 기능 요약

- 감정 일기
  - 텍스트 + 이모지 선택 + 이미지 첨부
  - 날짜별 작성/수정/삭제
- AI 해석
  - 일기 내용을 기반으로 감정 분석
  - AI 캐릭터가 오늘의 상태를 설명하는 코멘트 제공
- 시각화
  - 감정 점수, 이모지 통계를 기간별(주/월 등) 그래프로 표시
- 커뮤니티
  - 게시판, 댓글, 공감(이모지) 등
- 계정/프로필
  - 이메일/비밀번호 회원가입 및 로그인
  - 소셜 로그인(예: Kakao, Google, Naver 등)
  - MBTI, 생년월일, 프로필 이미지 설정

## 5. 기술 스택 개요

- **Frontend**
  - React, React Router
  - axios 기반 API 연동
- **Backend**
  - Spring Boot 3.x, JPA(Hibernate)
  - MySQL(RDS), Redis
- **Infra**
  - Docker, Docker Hub
  - AWS EC2, RDS, S3, (Route53, Nginx, Certbot 등)
