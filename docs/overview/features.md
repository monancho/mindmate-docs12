# MindMate 기능 상세

> 본 문서는 MindMate의 주요 기능을 기술적 관점에서 정리한 문서입니다.
> 기능 단위로 구조를 구분하여 서비스 동작 방식을 한눈에 이해할 수 있도록 구성하였습니다.

## 1. 감정 일기 기능

### 기능 개요

사용자가 텍스트·이모지·이미지를 활용해 하루의 감정을 기록할 수 있는 기능입니다. 기록된 정보는 AI 분석 및 시각화 모듈의 핵심 입력 데이터로 사용됩니다.

### 주요 기능 요소

-   텍스트 기반 일기 작성
-   AI가 작성한 일기 내용을 기반으로 감정 이모지 자동 선택
-   이미지 업로드 및 삭제
-   날짜 기반 CRUD
-   감정 점수·이모지 정보는 DB에 저장되며, AI 분석 및 시각화 기능에서 재활용

### 기술 포인트

-   이미지 업로드는 Multipart/FormData 기반 처리
-   작성·수정·삭제 시 상태 기반 UI 업데이트
-   감정 데이터는 통계 모듈 집계 대상

---

## 2. AI 감정 분석 기능

### 기능 개요

일기 텍스트를 기반으로 감정을 자동 분석하고, 사용자 상태를 요약한 코멘트를 제공합니다.

### 주요 기능 요소

-   일기 텍스트에서 감정 범주 자동 추출
-   감정 강도·변화(제미나이)를 활용한 결과 정리
-   작성 내용이 일기 형식이 아닐 경우 분석 제한
-   하루 상태 요약 코멘트 생성
-   분석 결과는 일기·게시판·시각화 기능과 연동

### 기술 포인트

-   복잡한 모델링이 아닌, **프롬프트 설계 기반**의 직관적 분석 흐름
-   일기/게시판 데이터와 연동되는 단순한 연결 구조
-   분석 실패 시 예외 응답 처리 포함

---

## 3. 시각화 리포트 기능

### 기능 개요

사용자의 감정 기록을 기간별 그래프·통계 형태로 시각화하여 감정 패턴을 직관적으로 확인할 수 있는 기능입니다.

### 주요 기능 요소

-   주간·월간 감정 변화 그래프
-   감정·이모지 통계 자동 집계
-   감정 패턴 하이라이트 제공
-   감정 통계 데이터를 CSV 파일로 내보내기

### 기술 포인트

-   React 기반 차트 라이브러리 사용
-   감정 분석 결과 및 일기 메타데이터 기반 집계 로직 적용
-   특정 기간 필터링 기능 지원

---

## 4. 심리테스트·오늘의 운세 기능

### 기능 개요

감정 기록 외에도 사용자 상태를 보조적으로 파악할 수 있는 간단한 테스트·별자리 운세 기능입니다.

### 주요 기능 요소

-   심리 성향 테스트
-   별자리 기반 오늘의 운세 제공
-   테스트 결과 요약

### 기술 포인트

-   테스트 로직은 프론트/백엔드 분리 설계
-   날짜·별자리 기반 결과 매핑
-   감정 기록과 독립적으로 동작

---

## 5. 커뮤니티 기능

### 기능 개요

사용자 간 상호작용을 지원하는 게시판 기능입니다.

### 주요 기능 요소

-   게시글 작성·조회·수정·삭제
-   댓글 및 공감 기능
-   게시글 검색
-   페이지네이션
-   게시글 내용을 기반으로 자동 해시태그 생성
-   생성된 해시태그를 활용한 인기 주제 모아보기

### 기술 포인트

-   게시글/댓글/공감 도메인 분리
-   검색(제목·내용·작성자) 필터링 지원
-   관리자 권한 기반 접근 제어
-   게시글 내용 기반 해시태그 추출 로직 적용
-   해시태그별 인기 게시글 집계

---

## 6. 계정·프로필 기능

### 기능 개요

사용자 계정 생성·로그인·설정 등을 관리하는 기능입니다.

### 주요 기능 요소

-   이메일 회원가입 및 로그인
-   카카오·네이버·구글 소셜 로그인
-   프로필 이미지·MBTI·생일 설정
-   비밀번호 변경 및 회원탈퇴

### 기술 포인트

-   JWT 기반 인증 구조 (Access Token + Refresh Token 분리)
-   Refresh Token + Redis 기반 자동 재발급 및 토큰 만료 관리
-   이메일 인증 코드는 Redis TTL 기반으로 관리
-   프로필 이미지는 S3 객체로 관리되며, 회원 탈퇴 시 연관 객체 정리
